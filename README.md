# 👻 Ghost Kitchen Simulator

**Ghost Kitchen Simulator** is a 3D cooperative simulation game developed in **Unity**.
The player starts as a delivery driver for an established food chain and gradually builds their own delivery company and restaurant business.

---

## Core Features

### Basic Gameplay

* Preparation of different dishes:

  * Pizza
  * Burger
  * Pasta
  * Salad
  * Drinks
* Order handling system
* Delivery of prepared dishes
  * Initial vehicle: Scooter or Bicycle
* Revenue-based progression system
* Business expansion mechanics
  1. Start as a delivery driver for an existing restaurant
  2. Deliver for multiple restaurants
  3. Open your own restaurant
  4. Expand and optimize

### Multiplayer Mode

* Shared kitchen and delivery tasks
* Dedicated server architecture
